FlashCoin is a non-profit, educational fork of Bitcoin Core. The goal of this project is to learn how a real cryptocurrency node and wallet work (consensus, networking, mining, RPC, GUI, etc.), not to build an investment product.

This document explains how you can help and how we handle contributions.

---

Project Scope
-------------

FlashCoin Core is currently a **small, experimental project** maintained by a very small team.

Because of that:

- The focus is on **learning, clarity, and stability**, not on adding every possible feature.
- Big or risky changes to consensus rules will almost always be rejected unless they are clearly documented and tested.
